This folder contains the set up for a reproducible workflow as described by https://github.com/FellowsFreiesWissen/computational_notebooks.git

The file structure is organized as such:

proj_name
|-- README.txt
|-- proj_name.jl
|-- results
|   |-- README.txt
|   |-- data
|   |   |-- raw
|   |   |-- processed
|   |   |-- metadata
|   |-- figures
|   |-- tables
|-- scripts
|-- tests
|-- text
|   |-- README.txt
|   |-- main.doc
|   |-- supplementary
|   |-- references
|-- submission
|   |-- README.txt
|   |-- journal1
|       |-- first
|   |-- journal2
|       |-- first
|       |-- revisions
